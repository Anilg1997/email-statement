How to Use - Statement Editor

1
Fill Details — Bank name, account holder, account number, etc.
2
Add Transactions — Click "Add Row" to add/edit transaction rows
3
Check Summary — Totals auto-calculate from your transactions
4
Download — Plain PDF (no password) or Encrypted PDF (for password-protected statements)

How to Use - Upload PDF

1
Go to Upload PDF tab
2
Enter the Account ID (must match Chrome Extension settings)
3
Drag & drop or click to select a PDF file
4
Click Upload — the PDF is stored and will be served via the replace endpoint

Password Info

The replace endpoint encrypts PDFs with AES-256 — just like real bank statements.

PDF Password: Last 4 digits of Account Number
Example: Account 9876543210 → Password 3210

Replace endpoint: GET /api/replace?accountId=XXXXX
Test it: /api/replace?accountId=1234567890
Opens with password: 7890

Chrome Extension Setup

1
Open Chrome → chrome://extensions/
2
Enable Developer mode (toggle top-right)
3
Click Load unpacked → select extension/ folder
4
Click extension icon → enter bank domain & account ID → Save
5
Done! Any PDF from your bank domain is now replaced with your custom statement
